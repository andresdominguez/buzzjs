# buzzjs
Sample test for buzzjs

## Instructions

```shell
# clone this repo
$ git clone https://github.com/andresdominguez/buzzjs.git
$ cd buzzjs
$ npm i protractor -g
$ webdriver-manager update
$ webdriver-manager start

# Open a new terminal
# Run all tests on chrome
$ protractor

# Run tests for firefox and chrome
$ protractor protractor-multibrowser.conf.js

# Run tests in parallel
$ protractor protractor-parallel.conf.js
```
